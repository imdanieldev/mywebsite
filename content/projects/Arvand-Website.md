# Arvand Financial Academy Website

Official website for **Arvand Financial Academy**, built with Nuxt 3 and TailwindCSS.  
This project provides a modern, multilingual platform showcasing services, agents, and academy information.

---

## Features
- **Multilingual Support**: Persian (Fa) and English (En) via i18n  
- **Core Pages**: Home, About Us, Contact Us, Services, Agents  
- **Modern UI/UX**: Responsive design system with TailwindCSS v4  
- **Nuxt 3 Architecture**: Modular, secure, and scalable  
- **Deployment**: Hosted on Netlify for fast global delivery  

---

## Tech Stack
- **Framework**: [Nuxt.js 3](https://nuxt.com/)  
- **Styling**: [TailwindCSS v4](https://tailwindcss.com/)  
- **Languages**: Vue (96.2%), CSS (2.4%), TypeScript (1.4%)  
- **Internationalization**: Nuxt i18n for locale management  
- **Deployment**: Netlify
